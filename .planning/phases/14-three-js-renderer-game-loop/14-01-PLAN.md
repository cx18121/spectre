## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WebSocket → useGameSocket | MsgFpsState and MsgFpsHit parsed from raw JSON; attacker controls server message content |
| MediaPipe worldLandmarks → Three.js | Keypoint coordinates from pose worker; values could be NaN/Infinity on ML failure |
| Three.js canvas | Renders to user's GPU; no user-supplied geometry in this plan |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-14-01-01 | Tampering | MsgFpsState parsing in useGameSocket | mitigate | Cast via `as unknown as MsgFpsState` after JSON.parse; add null-guard on hp array before indexing: `if (!Array.isArray(msg.hp) || msg.hp.length < 2) return` before setLastFpsState |
| T-14-01-02 | Denial of Service | renderer.setAnimationLoop infinite loop | mitigate | Cap dt at 50ms per tick (`Math.min(dt, 0.05)`) to prevent spiral-of-death if tab is backgrounded |
| T-14-01-03 | Information Disclosure | NaN keypoint propagates to Three.js | mitigate | In updateArmSegment, guard: `if (!isFinite(from.x) || !isFinite(to.x)) return` before applying position — prevents WebGL NaN corruption |
| T-14-01-04 | Denial of Service | Multiple GameRenderer mounts (React strict mode double-effect) | mitigate | Cleanup in useEffect return function: `renderer.setAnimationLoop(null); renderer.dispose(); container.removeChild(renderer.domElement)` — prevents renderer leak on remount |
