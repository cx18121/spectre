<objective>
Install the `1eurofilter` package and implement the two detection hooks that form the pose pipeline backbone: `usePose` (WCI-01) and `useOneEuroFilter` (WCI-02).

`usePose` is the fps/ adaptation of mobile's detection loop. The critical difference: it accepts the pre-warmed `workerRef` from `useWarmup` instead of spawning a new worker. It runs a rAF/rVFC capture loop, posts frames to the existing worker, and exposes `worldLandmarks` as React state.

`useOneEuroFilter` wraps the `1eurofilter` npm package for a `PoseKeypoint[]` array, maintaining 99 filter instances (33 landmarks × x/y/z axes) in a `useRef` so they remain stateful across renders.

Purpose: WCI-01 requires off-thread pose detection without dropping Three.js frame rate. WCI-02 requires OneEuroFilter smoothing to eliminate jitter false-positives that would trigger phantom punches.

Output: usePose.ts + tests, useOneEuroFilter.ts + tests, 1eurofilter installed.
</objective>

    ```typescript
    import { useRef } from 'react';
    import { OneEuroFilter } from '1eurofilter';
    import type { PoseKeypoint } from '@shared/protocol';

    export function useOneEuroFilter(
      keypoints: PoseKeypoint[] | null,
      freq = 60,
      mincutoff = 1.0,
      beta = 0.007,
      dcutoff = 1.0,
    ): PoseKeypoint[] | null {
      const filtersRef = useRef<Map<string, OneEuroFilter>>(new Map());
      const lastTsRef = useRef<number>(0);

      if (!keypoints) return null;

      const now = performance.now() / 1000; // seconds
      // Guard: OneEuroFilter requires strictly increasing timestamps
      const ts = now > lastTsRef.current ? now : lastTsRef.current + 1 / freq;
      lastTsRef.current = ts;

      const filters = filtersRef.current;
      return keypoints.map((kp, i) => {
        const result = { ...kp };
        for (const axis of ['x', 'y', 'z'] as const) {
          const key = `${i}_${axis}`;
          if (!filters.has(key)) {
            filters.set(key, new OneEuroFilter(freq, mincutoff, beta, dcutoff));
          }
          (result as Record<string, number>)[axis] =
            filters.get(key)!.filter(kp[axis] as number, ts);
        }
        return result;
      });
    }
    ```
