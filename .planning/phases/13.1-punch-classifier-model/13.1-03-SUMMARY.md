### Auto-fixed Issues

**1. [Rule 1 - Bug] Tensor mock used arrow function — cannot be called with `new`**
- **Found during:** Task 2 GREEN phase (Tests 5 and 6 failing)
- **Issue:** The plan's mock template used `vi.fn().mockImplementation((_type, data, dims) => ({ data, dims }))` — an arrow function. When the hook calls `new ort.Tensor(...)`, the arrow function throws `TypeError: not a constructor`
- **Fix:** Changed mock to use `function(_type, data, dims) { return { data, dims }; }` which can be called as a constructor
- **Files modified:** `fps/src/hooks/usePunchClassifier.test.ts`
- **Commit:** 64559c0
